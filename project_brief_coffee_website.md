# Project Brief: เว็บไซต์ร้านกาแฟไทย (ไว้ส่งต่อให้ Claude ทำเว็บ)

> เอกสารฉบับนี้คือสรุปทุกการตัดสินใจของเจ้าของแบรนด์ เพื่อใช้เป็น brief ส่งต่อให้ AI/นักพัฒนาทำเว็บไซต์ตามคอนเซ็ปต์ที่ตกลงกัน

---

## 1. ภาพรวมโปรเจกต์ (Project Overview)

### 1.1 เป้าหมายของเว็บไซต์
เว็บไซต์นี้เป็น **เว็บขาย ไม่ใช่เว็บแบรนด์โชว์เฉยๆ** ทำหน้าที่เป็น "หน้าร้านดิจิทัล" แทนหน้าร้านจริง เนื่องจากเจ้าของยังไม่มีหน้าร้านกายภาพ

**KPI หลักของเว็บ:**
- เปอร์เซ็นต์ผู้เข้าชมเว็บที่กดปุ่ม "สั่งซื้อ" ไปยังแพลตฟอร์มเดลิเวอรี (Grab / LINE MAN / Shopee Food)
- ทุกอย่างบนเว็บต้องเอื้อให้ผู้ใช้กดสั่งซื้อได้เร็วที่สุด

### 1.2 กลุ่มเป้าหมาย
- คนเมือง อายุ 22-40 ปี ที่ชอบกาแฟคุณภาพ
- สนใจเรื่องเล่า/ที่มาของวัตถุดิบ (Story-driven consumers)
- ใช้มือถือเป็นหลัก (Mobile-first audience)
- ยอมจ่ายแพงกว่าเพื่อความพิเศษและคุณภาพ

### 1.3 ข้อจำกัด/บริบทสำคัญ
- ไม่มีหน้าร้านจริง ทำกาแฟที่บ้าน/คอนโด
- ขายผ่าน Grab / LINE MAN / Shopee Food เท่านั้น
- งบประมาณภาพรวม 10,000-30,000 บาท
- ยังไม่ได้ถ่ายรูปสินค้าจริง ใช้ภาพ stock ฟรีไปก่อน

## 2. แบรนด์ดิ้ง (Brand Identity)

### 2.1 Brand Positioning
"กาแฟไทยพรีเมียม สไตล์ครัวอังกฤษผสมความอบอุ่นแบบไทย — ไม่เหมือนคาเฟ่ทั่วไปในตลาด"

### 2.2 Brand Personality
- **แปลกใหม่แต่ไม่ไฮเทคจนเย็นชา** (Distinctive but warm)
- **มีความหรูหรา แต่เข้าถึงได้** (Premium but approachable)
- **อบอุ่นแบบครัวบ้าน + เรียบหรูแบบอังกฤษ** (Cozy English kitchen aesthetic)
- **ภูมิใจในความเป็นไทย ผ่านเมล็ดกาแฟไทย** (Thai pride through Thai beans)

### 2.3 Brand Voice & Tone
- เล่าเรื่องอย่างนุ่มนวล มีรายละเอียด ชวนจินตนาการ
- ไม่ขายของแบบ hard-sell ไม่ใช้คำว่า "ลด" "โปร" "ถูก"
- ใช้คำพรรณนาความรู้สึก ("หอมกลมกล่อม" "อุ่นใจ" "สดชื่นเหมือนเช้าวันหยุด")
- เคารพรากเหง้าและคนปลูกกาแฟ

## 4. Web Architecture / โครงสร้างเว็บ

เว็บเป็น **single-page (one-page scroll)** เลื่อนยาวลงล่าง 6 section ตามลำดับ:

### Section 1: Hero (หน้าแรกเด่นที่สุด)
- ความสูงเต็มจอ (100vh)
- พื้นหลังภาพกาแฟใหญ่/อนิเมชันกาแฟหยด
- มี Logo + ชื่อแบรนด์ + tagline สั้นๆ
- **ปุ่ม "สั่งซื้อเลย" เด่นชัด ต้องเห็นภายในวินาทีแรก** (ไม่ต้องรออนิเมชันจบ)

### Section 2: Our Story (เราคือใคร)
- เล่าเรื่องสั้นๆ 2-3 ย่อหน้า
- คอนเซ็ปต์ "กาแฟไทยพรีเมียม ครัวอังกฤษผสมไทย"
- ภาพประกอบ: บรรยากาศการชง/บรรยากาศคาเฟ่

### Section 3: Our Beans (เมล็ดกาแฟไทย — ดอยตุง เชียงราย)
**ใช้เมล็ดดอยตุง จากจังหวัดเชียงราย**
- เรื่องราว: โครงการพระราชดำริ จากไร่ฝิ่นกลายเป็นไร่กาแฟ ชาวเขาเปลี่ยนอาชีพ
- ลักษณะรสชาติ: บอดี้นุ่ม คาราเมล ช็อกโกแลตอ่อน อัตติคาราเมล มะม่วงสุกอ่อน
- แสดง 3 ระดับการคั่ว:
  - **คั่วอ่อน (Light Roast):** เปรี้ยวอมหวาน กลิ่นดอกไม้ มะม่วง
  - **คั่วกลาง (Medium Roast):** สมดุล คาราเมล ถั่ว ช็อกโกแลตนม
  - **คั่วเข้ม (Dark Roast):** เข้มเต็มที่ ดาร์กช็อกโกแลต ติดควันอ่อนๆ

### Section 4: Menu (เมนู)
**เน้นเบสิคก่อน 6-7 เมนู** (ขยายได้ภายหลัง)
- Espresso — ช็อตเดี่ยว เข้มข้น
- Americano (ร้อน/เย็น) — กาแฟดำ สะอาด
- Latte (ร้อน/เย็น) — นมนุ่ม กลมกล่อม
- Cappuccino (ร้อน) — ฟองนมแน่น
- Mocha (เย็น) — ช็อกโกแลตเข้ม
- Matcha Latte — มัทฉะ (ฮิตช่วงนี้)
- Signature เมนูประจำร้าน (ยังไม่ระบุ คิดเพิ่มภายหลัง)

แต่ละเมนูควรมี: ชื่อ, รูป, คำพรรณนา 1 บรรทัด, ราคา, ปุ่มสั่งลิงก์ไปแพลตฟอร์ม

### Section 5: Order Now (สั่งซื้อ)
- 3 ปุ่มใหญ่: Grab Food | LINE MAN | Shopee Food
- ลิงก์ตรงไปร้านบนแต่ละแพลตฟอร์ม
- โลโก้แพลตฟอร์มชัด สีจริง

### Section 6: Footer
- ติดต่อ: Instagram, Facebook, LINE Official
- ขอบคุณคนปลูกกาแฟไทย
- Copyright

## 5. Animation & Interaction (อนิเมชัน)

**ระดับที่เลือก: เน้นเว่อ สวยจัด อาจช้านิดหน่อยได้**

### 5.1 Hero Animation
- กาแฟหยดลงแก้วเมื่อเข้าเว็บ (CSS animation 2-3 วินาที)
- ไอกาแฟลอยขึ้นแบบนุ่มนวล (วนซ้ำ)
- เมล็ดกาแฟลอยช้าๆ ในพื้นหลัง (parallax)

### 5.2 Scroll Animations
- แต่ละ section fade-in + slide-up เมื่อเลื่อนถึง (Intersection Observer + CSS)
- รูปกาแฟแก้วหมุนช้าๆ เมื่ออยู่ในวิวพอร์ต
- ตัวเลข/ราคา count-up จาก 0

### 5.3 Hover Interactions
- ปุ่มเปลี่ยนสี smooth (300ms)
- รูปเมนู zoom-in เบาๆ เมื่อ hover
- ใบไม้/เมล็ดกาแฟขยับเล็กน้อย

### 5.4 Beans Section (Special)
- แก้วกาแฟ 3 ใบ แทน 3 ระดับการคั่ว
- hover แต่ละแก้ว = เปลี่ยนสีกาแฟ (จาก light → dark) + แสดงรสชาติ

### **กฎสำคัญ:** อนิเมชันต้องไม่กั้นปุ่มสั่งซื้อ ผู้ใช้ต้องกดได้ตลอด ไม่ว่าอนิเมชันจะเล่นถึงไหน

## 6. Technical Requirements (เทคนิค)

### 6.1 เทคโนโลยีแนะนำ
- **Framework:** HTML/CSS/JS แบบเรียบง่าย หรือ React (ขึ้นกับใครทำ)
- **Animation library:** GSAP หรือ Framer Motion (ถ้าใช้ React)
- **Hosting:** Vercel หรือ Netlify (ฟรี)
- **Domain:** สั่ง .com ราคาประมาณ 400 บาท/ปี

### 6.2 Responsive Design
- **Mobile-first** (พฤติกรรมหลักของลูกค้า)
- รองรับ: มือถือ 360-430px, แท็บเล็ต 768px, เดสก์ท็อป 1280px+
- ปุ่มสั่งซื้อบนมือถือ: อย่างน้อย 48x48px (tap target)
- ฟอนต์ขั้นต่ำ 16px บนมือถือ

### 6.3 Performance
- Hero image โหลดเร็ว (lazy load รูปอื่น)
- ใช้ภาพ WebP หรือ AVIF
- อนิเมชัน CSS มากกว่า JS (เร็วกว่า)
- Page load < 3 วินาที (ปุ่มสั่งซื้อต้องโผล่ใน 1 วินาทีแรก)

### 6.4 Golden Ratio & Spacing
- ใช้ระยะห่างตามชุด 8px (8, 16, 24, 32, 48, 64, 96, 128)
- Heading ใหญ่: 64-80px desktop, 40-48px mobile
- Body text: 16-18px
- Section spacing: 96-128px desktop, 64px mobile
- Max content width: 1200px center aligned

## 9. ขั้นตอนการดำเนินงาน (Action Plan)

### Phase 1: เตรียมตัว (สัปดาห์ 1-2)
1. ตั้งชื่อร้าน + ออกแบบโลโก้
2. สั่งซื้อเมล็ดดอยตุง + ทดลองชง
3. สมัครเปิดร้านบน Grab, LINE MAN, Shopee Food
4. เตรียมเอกสาร: บัตร ปชช, บัญชีธนาคาร, รูปร้าน

### Phase 2: สร้างเว็บ (สัปดาห์ 3-4)
1. ส่งเอกสารนี้ + ข้อมูลที่เพิ่มเติม (ชื่อร้าน/โลโก้/ลิงก์) ให้ Claude
2. ให้ Claude สร้างเว็บตาม brief
3. ทดสอบบนมือถือ/เดสก์ท็อป
4. ปรับแก้ feedback รอบที่ 1-2

### Phase 3: ดีพลอย (สัปดาห์ 5)
1. สั่งโดเมน (.com)
2. ดีพลอยที่ Vercel/Netlify
3. ตั้งค่า Google Analytics
4. ทดสอบลิงก์ไปแพลตฟอร์มทั้งหมด

### Phase 4: เปิดตัว (สัปดาห์ 6)
1. โพสต์โซเชียลเปิดตัว
2. ส่งให้เพื่อนช่วยกระจาย
3. เก็บ feedback จากลูกค้าจริง 2-4 สัปดาห์
4. ปรับเว็บตามข้อมูลจริง

### Phase 5: ขยาย (เดือน 2-3 ขึ้นไป)
1. ถ่ายภาพสินค้าจริงแทน stock
2. เพิ่มหน้า Personality Quiz
3. เพิ่มเมนู Signature
4. พิจารณาขายเมล็ดกาแฟ/ของชำร่วย

## ภาคผนวก: คำถามที่ Cross-coach อาจถาม + คำตอบเตรียมไว้

**Q: ทำไมเลือกสไตล์ครัวอังกฤษผสมไทย ทั้งที่ขายในไทย?**
A: เพราะตลาดกาแฟไทยเต็มไปด้วยร้านสไตล์ญี่ปุ่น/มินิมอลขาวเทา การใช้โทนครัวอังกฤษ (warm, cozy, classic) จะทำให้แตกต่างและจดจำได้ ผสมความเป็นไทยผ่านวัตถุดิบ (ดอยตุง) และเรื่องเล่า ทำให้ดูพรีเมียมแต่ไม่หลุดราก

**Q: ทำไมต้องเป็นเว็บไซต์ ในเมื่อมี Grab/LINE MAN อยู่แล้ว?**
A: เว็บไซต์เป็น "ป้อมแบรนด์" ที่เราควบคุมได้ 100% ไม่ขึ้นกับอัลกอริทึมแพลตฟอร์ม ใช้เล่าเรื่องลึก สร้างความเชื่อมั่น และเป็นจุดศูนย์กลางที่กระจายลูกค้าไปแพลตฟอร์มต่างๆ ตามที่เขาสะดวก

**Q: อนิเมชันเว่อจะทำให้เว็บช้า ลูกค้าจะหนีไหม?**
A: อนิเมชันเว่อในจุด Hero เท่านั้น เพื่อสร้างความประทับใจแรก ส่วนล่างของหน้าใช้อนิเมชันเบาๆ เน้นความเร็ว และปุ่มสั่งซื้อทุกจุดต้องกดได้ทันที ไม่ติดอนิเมชัน

**Q: ทำไมเลือกดอยตุง ไม่ใช่ดอยช้าง/ดอยปางขอน?**
A: ดอยตุงมีเรื่องเล่าทรงพลังที่สุด (โครงการพระราชดำริ จากฝิ่นเป็นกาแฟ) ตรงกับ positioning "ภูมิใจในความเป็นไทย" และคนรู้จักดี ขายเรื่องราวได้ง่าย

**Q: งบ 10,000-30,000 พอจริงไหม?**
A: เพียงพอสำหรับเฟส 1 (อุปกรณ์ + วัตถุดิบ + บรรจุภัณฑ์) โดเมน 400 บาท โฮสต์ฟรี ค่า Grab/LINE MAN ฟรี เก็บค่าคอม 30% จากยอดขายแทน
